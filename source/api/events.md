谢岗福易是一家专业经销不锈钢品牌门窗、实木门窗，及生态门窗的等厂家指定代理经销商，所有产品均由品牌门窗厂家直接供货。经销范围覆盖谢岗镇及辖区，以及常平镇、樟木头镇、清溪镇、沥林镇和桥头镇等谢岗周边镇区。
